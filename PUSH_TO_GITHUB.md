# Instructions to Push to GitHub

## Step 1: Create Repository on GitHub
1. Go to: https://github.com/new
2. Repository name: `health_project_Surpervised`
3. Choose Public or Private
4. **DO NOT** check "Initialize this repository with a README"
5. Click "Create repository"

## Step 2: Run these commands in your terminal

```bash
git remote add origin https://github.com/sanjanaramgarhia/health_project_Surpervised.git
git branch -M main
git push -u origin main
```

Note: If you named your repository differently, replace `health_project_Surpervised` in the URL above with your repository name.
